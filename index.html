Mastering HTML and CSS

HTML Fundamentals

HTML (HyperText Markup Language) is the foundation of all web pages. It structures the content on the web. Start by learning the basic structure:

<!DOCTYPE html>
<html>
    <head>
        <title>My First Page</title>
    </head>
    <body>
        <h1>Hello, World!</h1>
        <p>This is my first webpage.</p>
    </body>
</html>

Basic HTML Elements: Learn the most common HTML tags like <p>, <a>, <div>, and <img>. Attributes, such as src for images and href for links, provide additional information about the elements.

Semantic HTML: Use semantic tags like <header>, <footer>, <article>, and <section> to improve the accessibility and readability of your HTML code.

CSS Fundamentals

Introduction to Node.js

Node.js is a JavaScript runtime that allows you to run JavaScript on the server. It's known for its event-driven, non-blocking I/O model, which makes it efficient and scalable.

  • Setting Up Node.js: Install Node.js and npm (Node Package Manager) on your machine.
  • Creating a Simple Server: Write a basic Node.js server that listens for HTTP requests.
const http = require('http');
        
        const server = http.createServer((req, res) => {
            res.statusCode = 200;
            res.setHeader('Content-Type', 'text/plain');
            res.end('Hello, World!\n');
        });
        
        server.listen(3000, () => {
            console.log('Server running at http://localhost:3000/');
        });

Basic CRUD Operations

CRUD (Create, Read, Update, Delete) operations are fundamental to working with databases.

  • Create: Insert new records into the database.
  • Read: Retrieve records from the database.
  • Update: Modify existing records.
  • Delete: Remove records from the database.
-- Create a new table
        CREATE TABLE users (
            id INT AUTO_INCREMENT PRIMARY KEY,
            name VARCHAR(100),
            email VARCHAR(100)
        );
        
        -- Insert a new record
        INSERT INTO users (name, email) VALUES ('John Doe', 'john@example.com');
        
        -- Retrieve records
        SELECT * FROM users;
        
        -- Update a record
        UPDATE users SET email = 'john.doe@example.com' WHERE id = 1;
        
        -- Delete a record
        DELETE FROM users WHERE id = 1;

Connecting Backend to Database

Learn how to connect your backend application to a database and perform CRUD operations.

  • Using ORMs/ODMs: ORMs (Object-Relational Mappers) like Sequelize and ODMs (Object-Document Mappers) like Mongoose make it easier to interact with databases in a programmatic way.
const { Sequelize, DataTypes } = require('sequelize');
        const sequelize = new Sequelize('database', 'username', 'password', {
            host: 'localhost',
            dialect: 'mysql'
        });
        
        const User = sequelize.define('User', {
            name: {
                type: DataTypes.STRING,
                allowNull: false
            },
            email: {
                type: DataTypes.STRING,
                allowNull: false
            }
        });
        
        sequelize.sync().then(() => {
            return User.create({
                name: 'John Doe',
                email: 'john@example.com'
            });
        });
